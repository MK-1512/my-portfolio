## üß† Notes & Usage Tips

- The **EmailJS** integration requires environment variables. Create a `.env` file and add your keys as:  
  ```bash
  VITE_EMAILJS_SERVICE_ID=your_service_id
  VITE_EMAILJS_TEMPLATE_ID=your_template_id
  VITE_EMAILJS_PUBLIC_KEY=your_public_key
  ```  
- For best performance, deploy using **Vercel** or **Netlify** (Vercel recommended).  
- Tailwind automatically purges unused CSS during build for optimized file size.
